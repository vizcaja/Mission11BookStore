# Mission 12 Bookstore

This project is a full-stack bookstore application using:
- ASP.NET Core Web API
- SQLite Database
- React + TypeScript
- Bootstrap

## Features
- Category filtering
- Pagination
- Sorting (A-Z, Z-A)
- Add to Cart
- Cart Summary
- Cart Page
- Update Quantity
- Remove Items
- Continue Shopping
- Cart stored in sessionStorage

## How to Run Backend
1. Open backend project in Visual Studio
2. Make sure Bookstore.sqlite is set to:
   - Build Action = Content
   - Copy to Output Directory = Copy if newer
3. Run the backend

## How to Run Frontend
1. Open terminal in frontend folder
2. npm install
3. npm run dev
